Presentation
------------

The aim of this Guide is to quickly give the way to customise PacketFence Portals and Emails.
This documentation will gones in complement of the _Developers Guide_.

Portal Customisation
--------------------

Portals are made of severals modules, depending of configured authentication methodes, ut the first module is always the *Root* modules.
This root module is _html_ coded using a template called the *layout.htlm*.
This layout will be modified will style included in the css configuration. 

Colors Customisation
~~~~~~~~~~~~~~~~~~~~

The Portals are coded and compiled with https://github.com/inuitcss[InuitCSS], this will allow us to change colors with css, in a template.

|===

| *_settings.packetfence.scss*  | Colors to be compiled in css.

| $inuit-base-text-color | Based text like User Name, Email, 

| $inuit-base-background-color:     |  Page background (Check INFO)  

| $inuit-btn-background:     |  Buttons and icons background

| $pf-btn-background-light:     |   Buttons color 

| $pf-base-accent-color:     |   Dots and  Icons background color

| $pf-footer-background-color:     |    

| $pf-footer-text-color:     |    

| $pf-box-tint-background:     |    

| $pf-box-tint-color:     |    

| $pf-frame-background:     |    

| $pf-list-tint-background:     |    

| $pf-list-tint-color:     |    

| $pf-list-background-light:     |    

| $pf-list-icon-color:     |    

| $pf-input-caption:     |    

| $pf-icon-symbole-color:     |  Icon color (Shield, Key, ...)

| $pf-icon-background-color   |   Icon Background color (Shield, Key, ...)

|===
INFO: To be able to see the page backgroup color, you need to replace  *<div class="layout layout--center frame--earlyMorning">* with  *<div class="layout layout--center">*, in /usr/local/pf/html/captive-portal/templates/layout.html

SCSS Files list
+++++++++++++++

|===

| *_components.iconic.scss*  | Icons grapical properties

| *_base.headings.scss* | h1;h2;h3 colors.

Most of the colors are in the *_settings.packetfence.scss*, in */usr/local/pf/html/common/scss*

You can edit *_settings.packetfence.scss* with vi, included in the packages, or by installing vim.
